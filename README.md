# Calculadora em Java (Swing / NetBeans)

Projeto de **calculadora gráfica** desenvolvido em **Java Swing**, criado no **NetBeans**. A aplicação possui interface visual (GUI) e executa operações matemáticas básicas por meio de botões, sem uso de terminal. O projeto utiliza arquivos `.form`, portanto foi estruturado para ser aberto e executado diretamente no NetBeans.

## Tecnologias
- Java (JDK 8 ou superior)
- Java Swing
- NetBeans IDE

## Como executar no NetBeans
- Extraia o arquivo `.zip`
- Abra o NetBeans
- Vá em `File > Open Project`
- Selecione a **pasta raiz do projeto** (a que contém `src` e `nbproject`)
- Clique com o botão direito no projeto e selecione **Run**
- Confirme a classe `main` se solicitado

## Executar fora do NetBeans
- No NetBeans: botão direito no projeto > **Clean and Build**
- Vá até a pasta `dist/`
- Execute no terminal:
  ```bash
  java -jar NomeDoProjeto.jar
